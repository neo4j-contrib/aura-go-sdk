## v1.11.0 - 2026-05-12
### Added
* AuraAPIClient.Close() drains idle HTTP connections held by the underlying transport. Call via defer after NewClient to avoid leaking file descriptors.
* Three new functional options: WithHTTPClient injects a custom *http.Client transport; WithUserAgent overrides the default User-Agent header; WithDefaultHeaders sets additional headers on every API request (Authorization, Content-Type, and User-Agent are silently ignored to prevent credential leakage).
### Changed
* Renamed CmekService to CMEKService, client.Cmek to client.CMEK, GetCmeksResponse to GetCMEKsResponse, and GetCmeksData to GetCMEKsData to follow Go acronym naming conventions. This is a breaking change.
* Renamed QueryMetrics.QueriesPerSecond to QueryExecutionTotal to accurately reflect that the value is a cumulative counter (neo4j_db_query_execution_success_total), not a per-second rate. JSON tag updated from `queries_per_second` to `query_execution_total`. This is a breaking change.
### Fixed
* CMEKService.List now sends `?tenant_id=` (not `?tenantID=`) so the Aura API correctly applies the tenant filter
* GetMetricValue now logs at Debug level (not Error) when a metric name is not found. Missing metrics are expected on certain Neo4j plans and versions and are not error conditions.
* OverwriteFromSnapshot now validates that sourceSnapshotID is a valid UUID, consistent with OverwriteFromInstance. Previously only an empty string was rejected.
