# Changelog

All notable changes to this project will be documented in this file.

The format is based on [changie](https://changie.dev) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## v1.0.0 - 2026-05-12

Initial release as `github.com/neo4j-contrib/aura-go-sdk`.

This is a continuation of `github.com/LackOfMorals/aura-client` (archived at v1.11.1) under the Neo4j organisation. The module path has changed; all other public API is identical.

### Changed

* Module path renamed from `github.com/LackOfMorals/aura-client` to `github.com/neo4j-contrib/aura-go-sdk`
* Default User-Agent header updated from `aura-go-client/<version>` to `aura-go-sdk/<version>`
