# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## v1.3.0 - 2026-06-17
### Added
* Add v2beta1 sub-package with Organizations and Projects services for managing Aura organizations and their projects via the v2beta1 API

## v1.2.0 - 2026-06-17
### Changed
* Revert Go module path from `github.com/neo4j-contrib/aura-go-sdk/v2` to `github.com/neo4j-contrib/aura-go-sdk`; the `/v2` suffix was added prematurely and conflicts with the planned Aura API v2 sub-package strategy
* Align CreateInstanceConfigData with the POST /instances API spec: remove CDCEnrichmentMode and SecondariesCount (PATCH-only fields), add CustomerManagedKeyID, make Version required (no omitempty), change VectorOptimized and GraphAnalyticsPlugin from pointer to value types; fix CreateFromSnapshot to require both sourceInstanceID and sourceSnapshotID per spec
### Added
* Add IsNotFound(err error) bool helper to detect Aura API 404 responses without importing internal packages
* Add CreatedAt *time.Time field to InstanceData
* Add WithMaxResponseSize option to configure the maximum response body size (default 10MB)

## v1.1.0 - 2026-05-12
### Added
* Add CDCEnrichmentMode, SecondariesCount, VectorOptimized, and GraphAnalyticsPlugin fields to CreateInstanceConfigData; add CDCEnrichmentMode and SecondariesCount fields to UpdateInstanceData; add CreateFromInstance and CreateFromSnapshot methods to InstanceService

## v1.0.0 - 2026-05-12
### Changed
* Module path renamed from `github.com/LackOfMorals/aura-client` to `github.com/neo4j-contrib/aura-go-sdk`
* Default User-Agent header updated from `aura-go-client/<version>` to `aura-go-sdk/<version>`
