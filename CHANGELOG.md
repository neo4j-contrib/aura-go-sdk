## v2.0.0 - 2026-05-12
### Changed
* Align CreateInstanceConfigData with the POST /instances API spec: remove CDCEnrichmentMode and SecondariesCount (PATCH-only fields), add CustomerManagedKeyID, make Version required (no omitempty), change VectorOptimized and GraphAnalyticsPlugin from pointer to value types; fix CreateFromSnapshot to require both sourceInstanceID and sourceSnapshotID per spec

## v1.1.0 - 2026-05-12
### Added
* Add CDCEnrichmentMode, SecondariesCount, VectorOptimized, and GraphAnalyticsPlugin fields to CreateInstanceConfigData; add CDCEnrichmentMode and SecondariesCount fields to UpdateInstanceData; add CreateFromInstance and CreateFromSnapshot methods to InstanceService

## v1.0.0 - 2026-05-12
### Changed
* Module path renamed from `github.com/LackOfMorals/aura-client` to `github.com/neo4j-contrib/aura-go-sdk`
* Default User-Agent header updated from `aura-go-client/<version>` to `aura-go-sdk/<version>`
